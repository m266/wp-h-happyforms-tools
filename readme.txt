== Description ==
Links in multi selection fields / Links in Mehrfachauswahl-Felder

== Installation ==
1. Upload 'wp-h-happyforms-tools'  to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
